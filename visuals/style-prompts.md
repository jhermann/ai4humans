# Short & Sweet Style Prompts

Add these to an image generation request or similar,
to influence the visual appearance.

- **"Notion-style" (Handcrafted 2D)**: <br /> Flat 2D vector, Notion style line art, hand-drawn ink strokes, minimalistic style, lo-fi aesthetic. Keep it airy, use enough white space, but also balance it with the provided level of detail.

  > ![style example](../assets/images/notion-style_720x393.png) *Gemini 3 Flash (Nano Banana 2) with "Enamel pin."*

- **"Roblox-style" (Low-poly 3D Game World)**: <br /> Roblox-style low-poly 3D world, blocky cartoon avatars mid-action, vibrant high-contrast palette (bright blues, greens, yellows, reds). Chunky bold gaming typography, pixelated UI elements, neon-lit in-game environments (cityscapes, floating platforms, lobbies). Characters with floating name tags and speech bubbles; expressive poses — pointing, gesturing, holding signs.

  > ![style example](../assets/images/roblox-style_720x393.png) *Gemini 3 Flash (Nano Banana 2) with "Technicolor."*
