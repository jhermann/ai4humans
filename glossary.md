## A

### Agentic AI
*Agentic AI* describes AI systems that can pursue goals across multiple steps, make decisions along the way, and use tools with limited human guidance.

> *See also:* [Agentic AI (Wikipedia)](https://en.wikipedia.org/wiki/Agentic_AI)

### Artificial General Intelligence (AGI)
*Artificial General Intelligence (AGI)* is a theoretical type of AI that could handle most mental tasks at roughly human level across many domains.

> *See also:* [Artificial general intelligence (Wikipedia)](https://en.wikipedia.org/wiki/Artificial_general_intelligence)

### Artificial Superintelligence (ASI)
*Artificial Superintelligence (ASI)* is a hypothetical form of AI that would outperform humans in nearly all cognitive tasks.

> *See also:* [Superintelligence (Wikipedia)](https://en.wikipedia.org/wiki/Superintelligence)

## F

### Fine-tuning
*Fine-tuning* is extra training on a pre-trained model so it performs better for a specific task or domain.

> *See also:* [Fine-tuning (deep learning) (Wikipedia)](https://en.wikipedia.org/wiki/Fine-tuning_(deep_learning))

## L

### Large Reasoning Models
*Large Reasoning Models* are AI models tuned to solve complex problems by working through intermediate reasoning steps before producing a final answer.

> *See also:* [Chain-of-thought prompting (Wikipedia)](https://en.wikipedia.org/wiki/Chain-of-thought_prompting)

### Large Language Model (LLM)
*Large Language Model (LLM)* is an AI model trained on large amounts of text so it can understand and generate human-like language.

> *See also:* [Large language model (Wikipedia)](https://en.wikipedia.org/wiki/Large_language_model)

## M

### Mixture of Experts (MoE)
*Mixture of Experts (MoE)* is a model design where only a small subset of specialized model parts is activated for each request, which can improve efficiency.

> *See also:* [Mixture of experts (Wikipedia)](https://en.wikipedia.org/wiki/Mixture_of_experts)

### Model Context Protocol (MCP)
*Model Context Protocol (MCP)* is a standard way for AI systems to connect to external tools and data sources, so one assistant can work with many integrations through the same pattern.

> *See also:* [Communication protocol (Wikipedia)](https://en.wikipedia.org/wiki/Communication_protocol)

## P

### Prompt Injection
*Prompt Injection* is a security attack where hidden or malicious instructions in input try to make an AI system ignore its intended rules.

> *See also:* [Prompt injection (Wikipedia)](https://en.wikipedia.org/wiki/Prompt_injection)

## R

### Retrieval-Augmented Generation (RAG)
*Retrieval-Augmented Generation (RAG)* is a method where a model first looks up relevant documents, then uses that information to produce a more grounded answer.

> *See also:* [Retrieval-augmented generation (Wikipedia)](https://en.wikipedia.org/wiki/Retrieval-augmented_generation)

## V

### Vector Database
*Vector Database* is a database built for storing and searching vector embeddings so systems can find content by semantic similarity instead of exact keyword matching.

> *See also:* [Vector database (Wikipedia)](https://en.wikipedia.org/wiki/Vector_database)
