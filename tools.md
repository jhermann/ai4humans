## Favorites

### GitHub Copilot
*GitHub Copilot* is an AI coding assistant that helps developers write, understand, test, and review code faster across IDEs, terminal workflows, and GitHub itself.

**Core features:**

- Inline code completions while typing in supported IDEs.
- Chat-based coding help for explanations, refactors, and implementation guidance.
- Terminal assistance via Copilot CLI for natural-language command workflows.

**Links:**

- *Homepage:* [GitHub Copilot: AI coding assistant](https://github.com/features/copilot)
- *Documentation:* [GitHub Copilot docs](https://docs.github.com/en/copilot)
- *Community Reaction:* [Is GitHub Copilot Getting Worse in 2026?](https://www.nxcode.io/resources/news/github-copilot-getting-worse-2026-developers-switching)

### Google Gemini
*Google Gemini* is Google's multimodal AI assistant and model family for chat, reasoning, coding, content generation, and workflow automation across web and Google Workspace products.

**Core features:**

- Multimodal input and output across text, images, and files.
- Strong reasoning and coding support for analysis, planning, and implementation help.
- Deep integration with Google ecosystem tools like Docs, Gmail, and Drive.

**Links:**

- *Homepage:* [Gemini](https://gemini.google.com/)
- *Documentation:* [Google AI for Developers - Gemini API docs](https://ai.google.dev/gemini-api/docs)

### Google NotebookLM
*Google NotebookLM* is an AI research and synthesis workspace that grounds answers in user-provided sources, helping teams and individuals study, summarize, and generate insights from trusted documents.

**Core features:**

- Source-grounded Q&A using uploaded documents, notes, and links.
- Automatic summaries, topic extraction, and key-point synthesis.
- Citation-aware responses that trace outputs back to source material.

**Links:**

- *Homepage:* [NotebookLM](https://notebooklm.google/)
- *Overview:* [Google NotebookLM Help](https://support.google.com/notebooklm)
