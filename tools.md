## Favorites

### GitHub Copilot
GitHub Copilot is an AI coding assistant that helps developers write, understand, test, and review code faster across IDEs, terminal workflows, and GitHub itself.

**Core features:**

- Inline code completions while typing in supported IDEs.
- Chat-based coding help for explanations, refactors, and implementation guidance.
- Terminal assistance via Copilot CLI for natural-language command workflows.

**Links:**

- *Homepage:* [GitHub Copilot: AI coding assistant](https://github.com/features/copilot)
- *Documentation:* [GitHub Copilot docs](https://docs.github.com/en/copilot)
- *Community Reaction:* [Is GitHub Copilot Getting Worse in 2026?](https://www.nxcode.io/resources/news/github-copilot-getting-worse-2026-developers-switching)

### Google Gemini
Google Gemini is Google's multimodal AI assistant and model family for chat, reasoning, coding, content generation, and workflow automation across web and Google Workspace products.

**Core features:**

- Multimodal input and output across text, images, and files.
- Strong reasoning and coding support for analysis, planning, and implementation help.
- Deep integration with Google ecosystem tools like Docs, Gmail, and Drive.

**Links:**

- *Homepage:* [Gemini](https://gemini.google.com/)
- *Documentation:* [Google AI for Developers - Gemini API docs](https://ai.google.dev/gemini-api/docs)

### Google NotebookLM
Google NotebookLM is an AI research and synthesis workspace that grounds answers in user-provided sources, helping teams and individuals study, summarize, and generate insights from trusted documents.

**Core features:**

- Source-grounded Q&A using uploaded documents, notes, and links.
- Automatic summaries, topic extraction, and key-point synthesis.
- Citation-aware responses that trace outputs back to source material.

**Links:**

- *Homepage:* [NotebookLM](https://notebooklm.google/)
- *Overview:* [Google NotebookLM Help](https://support.google.com/notebooklm)

## New

### Blender MCP
This tool connects a professional 3D design program to a smart chat window so anyone can build digital worlds. It follows instructions in plain English to place objects and adjust lighting without the user needing to learn complex buttons. This allows beginners to create architectural layouts or game assets in seconds.

**Links:**

- *Homepage:* [Blender MCP Repository](https://github.com/pali/blender-mcp)
- *Documentation:* [Model Context Protocol Introduction](https://modelcontextprotocol.io/introduction)
- *Community Reaction:* [A Markdown File Just Replaced Your Most Expensive Design Meeting](https://www.youtube.com/watch?v=D-Zp2fI5b7k)

### Claude Code
This is a programming helper that lives in your computer's command center and performs tasks by writing and running code. It can read design files to build websites and can be set to run automatically on a schedule. It acts as a central brain that can use other specialized tools to create videos or update documentation.

**Links:**

- *Homepage:* [Claude.ai by Anthropic](https://www.anthropic.com/claude)
- *Documentation:* [Claude Code Guide](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code)
- *Community Reaction:* [Using Claude Code for Everything](https://www.youtube.com/watch?v=vV2p_Vz8K1Y)

### Hyper 3D
This service takes a written description of a character or object and turns it into a three-dimensional model. These models can be dropped directly into a digital scene to make it more detailed. It helps people populate their 3D worlds without having to sculpt items by hand.

**Links:**

- *Homepage:* [Hyper 3D: Text to 3D](https://www.hyper.ai/)
- *Documentation:* [Hyper 3D API Docs](https://docs.hyper.ai/)
- *Community Reaction:* [Hyper 3D: Generative AI for 3D Assets](https://twitter.com/hyper_3d)

### Polyhaven
This is a digital library that provides high-quality surfaces and 3D objects for free. It helps creators add realistic details like textures and lighting to their virtual environments. The service is built directly into newer AI design systems to make finding assets easier.

**Links:**

- *Homepage:* [Polyhaven: The Public 3D Asset Library](https://polyhaven.com/)
- *Documentation:* [Polyhaven FAQ](https://polyhaven.com/faq)
- *Community Reaction:* [Why Polyhaven is the Best for 3D Artists](https://www.youtube.com/watch?v=Nn_E_XlVf-k)

### Remotion
This framework builds videos by using code instead of a traditional editing timeline. Because the video is made of code, you can easily change one detail and have the entire video update automatically. It is perfect for making professional-looking product demos or data charts very quickly.

**Links:**

- *Homepage:* [Remotion: Create Videos in React](https://www.remotion.dev/)
- *Documentation:* [Remotion Documentation](https://www.remotion.dev/docs)
- *Community Reaction:* [Video as Code: The Future of Production?](https://www.youtube.com/watch?v=avgZ0U6pGUM)

### SketchFab
This online platform hosts a massive collection of 3D models created by people around the world. Designers use it to find specific items to put into their virtual scenes without making them from scratch. It works with AI tools to let users search for and drop objects into a workspace using simple commands.

**Links:**

- *Homepage:* [Sketchfab: Publish & Find 3D Models](https://sketchfab.com/)
- *Documentation:* [Sketchfab Help Center](https://help.sketchfab.com/hc/en-us)
- *Community Reaction:* [Sketchfab vs ArtStation for 3D Artists](https://www.youtube.com/watch?v=N-B5r7H8A_U)

### Stitch
This tool from Google lets you describe an app or website, and it immediately draws the finished screens for you. It can handle voice commands and understands the context of your whole project as you make changes. It creates high-quality layouts with colors and fonts that are ready for developers to use.

**Links:**

- *Homepage:* [Google Stitch via Google Labs](https://labs.google/)
- *Documentation:* [Google Stitch Product Info](https://labs.google/)
- *Community Reaction:* 🎦 [Google Stitch: The AI Design Tool Taking On Figma](https://www.youtube.com/watch?v=Dk0dSiEke0M)
